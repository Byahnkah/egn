No arquivo index.html é necessário carregar, com a tag script, o arquivo egn.js e o arquivo no qual será colocado o código. No caso do exemplo, é o myBricks.js. Nele são chamadas duas funções: moveEverything() e drawEverything(). A função moveEverything() é responsável pela movimentação das entidades e deve-se chamar nela os métodos das classes dos elementos que possuem alguma interação, bem como as colisões dos objetos. Já a função drawEverything(), desenha as entidades na tela e carrega a imagem a ser aplicada na bola. Essa imagem pode ser substituída pelo arquivo em png na pasta, mantendo o mesmo nome e formato de imagem. Outros elementos podem ser desenhados em tela aravés dessa função, é necessário para isso chamar a classe Utils e o método dela que se quer utilizar, por exemplo: Utils.colorRect(posX, posY, largura, altura, cor). As entidades recebem suas respectivas classes e os parâmetros delas podem ser modificados.
